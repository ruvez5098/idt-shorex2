## 🚀 Quick Command Reference

### One-line start:
```bash
npm install && npm run dev:all
```

### Detailed steps:
```bash
# Install
npm install

# Start frontend (Terminal 1)
npm run dev

# Start backend (Terminal 2)
npm run server

# Or both together
npm run dev:all

# Train model (optional)
python train_plastic_model.py --epochs 50
```

### Access:
- Desktop: http://localhost:3000
- Phone: http://YOUR_IP:3000

## 📱 Using on Phone

1. **Get your IP**: 
   - Windows: `ipconfig`
   - Mac/Linux: `ifconfig`

2. **On phone, go to**: 
   ```
   http://YOUR_IP:3000
   ```

3. **Tap "Plastic AI"** at bottom

4. **Tap "Start Detection"**

5. **Allow camera access**

6. **Point at plastics!** 🎯

## 📚 Key Concepts Explained

### What is this?
A system that detects plastics in real-time using your phone camera and AI.

### How does it work?
1. Phone camera captures video
2. React app processes frames
3. TensorFlow.js AI detects plastics
4. Bounding boxes show detected items
5. All happens on your phone (no server needed)

### What do I need?
- Node.js (for running)
- Python (optional, for training custom models)
- A phone with a browser
- Good lighting for best results

### Can I train my own model?
Yes! Use `python train_plastic_model.py` with your own images.

### Does it work offline?
Yes, after the initial model loads, everything works offline.

### Is my data private?
Yes! All processing happens on your device. Nothing is sent anywhere.
